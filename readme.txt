Чтобы запустить клиент, необходимо:

1. Запустить сервер командой node server.js
2. Открыть index.html в браузере

Для демонстрации работы клиента:

1. Вписать любое из 4 кодовых слов: keyword1, keyword2, keyword3, keyword4. Тогда сайт покажет файлы URL.
2. Если вписать любое другое слово, сайт выдаст ошибку.

Так же на сайте присутствует счетчик загрузки, который показывает в процентах насколько загрузился файл. Обем и количество потоков не удалось реализовать.
